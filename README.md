# SM345 (성공회대학교 SM멘토링 사업 데이터베이스 캡스톤디자인)

## About

SM사업이란? 

소프트웨어공학과 멘토링 사업이라는 말로 성공회대 소프트웨어공학과 내에서 선후배간 지식 공유가 이루어지는 것을 의미한다.

하이브리드 앱 어플리케이션을 개발하여 멘토링 사업을 진행합니다.
